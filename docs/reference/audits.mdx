# Audits

The Rarimo Protocol components and smart contracts have undergone independent security auditing. All issues found during these audits have been acknowledged and/or fixed.

The following companies conducted the audits:

- [Halborn](https://www.halborn.com/)
- [Ambisafe](https://ambisafe.com/)

## Public Reports

### RariMe Snap

**Date**: 2023-08-28

**Scope**: https://github.com/rarimo/rarime

**Auditor**: [Halborn](https://www.halborn.com/)

**Results**: _2 medium and 1 low issues solved, 1 informational issue acknowledged_
