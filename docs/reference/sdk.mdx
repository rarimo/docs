# Rarimo SDK

Rarimo's JavaScript/TypeScript SDK provides tools to interact with browser-based wallets and send transactions from wallets to Rarimo's contracts.

For complete reference information about the SDK packages, see the [SDK reference](https://rarimo.github.io/js-sdk/index.html).

## Packages

The SDK provides these groups of packages:

- [Packages that provide access to wallets](./sdk/packages-provider)
- [Packages that send transactions to blockchains](./sdk/packages-bridge)
- [Packages that provide resources for other packages](./sdk/packages-base)

## Prerequisites

The SDK requires the [`ethers`](https://www.npmjs.com/package/ethers) package.

## Installation

To install the SDK, run `yarn add` and the names of the packages to install, as in this example:

```shell
yarn add ethers @rarimo/provider @rarimo/providers-evm @rarimo/nft-checkout
```
